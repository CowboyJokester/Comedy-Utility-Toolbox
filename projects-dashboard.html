How to Use This Tool

Purpose

The Projects Dashboard helps you store comedy projects, add bits inside each project, and track performance history so your material stays organized instead of scattered across napkins, notebooks, and mysterious voice memos.

When To Use It

  • When you want to organize several bits under one larger project.
  • When you need to track bit status, runtime, notes, and live performance results.
  • When you want your writing notes and show results in one place.

How To Use It

  1. Create a project name and choose a category.
  2. Add general project notes.
  3. Open the saved project card.
  4. Add individual bits inside that project.
  5. Add performance logs after live shows.
  6. Review what is working, what needs rewriting, and what should move into a setlist.

Example Input

Project: Grandpa Stories
Bit: Grandpa Refuses GPS
Performance Note: Worked well with older crowd; tighten the middle.

Example Output

Organized Result: One saved project containing bits, notes, runtimes, and live performance history.

Comedy Tip

Update performance notes soon after the show. Your memory lies after midnight and becomes a full-blown fiction writer by breakfast.
