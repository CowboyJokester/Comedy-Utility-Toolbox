How to Use This Tool

Purpose

The Setlist Architect helps you pull saved bits from your comedy projects, arrange them into running order, estimate runtime, save setlists, and print a clean performance copy.

When To Use It

  • When preparing an open mic, club set, clean show, corporate set, veteran show, festival, or showcase.
  • When you need to test different openers, middles, closers, and running orders.
  • When you want to know how long your set will run before you walk onstage.

How To Use It

  1. Create bits inside your Projects area first.
  2. Open Setlist Architect.
  3. Name the setlist and choose the show type.
  4. Add notes about audience, venue, opener, or closer.
  5. Add available bits to the current set.
  6. Move bits up or down into the strongest order.
  7. Save or print the finished setlist.

Example Input

Setlist Name: Tight 5 for Friday Night
Show Type: Comedy Club
Bits: Opener, Grandpa GPS, Aldi Cart, Callback Closer

Example Output

Finished Set: A saved and printable running order with estimated runtime.

Comedy Tip

Build a set by energy, not just topic. A good set has a clear opener, a building middle, and a closer that feels intentional.
