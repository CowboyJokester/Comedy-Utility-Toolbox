Hidden Assumption Filter

Find invisible assumptions hiding inside premises.

Need help finding the invisible assumption inside a premise?

How to Use This Tool

Purpose

The Hidden Assumption Filter helps comedians uncover the invisible beliefs, expectations, and unspoken rules hiding underneath a premise.

When To Use It

Use this tool when a premise feels obvious, flat, too familiar, or stuck. It helps reveal the quiet belief people accept as normal so you can challenge it, reverse it, or exaggerate it.

How To Use It

  1. Enter a comedy premise or observation.
  2. Click Analyze Assumptions.
  3. Review the surface assumptions.
  4. Use the pressure questions to find conflict.
  5. Look for pride, fear, control, status, comfort, or identity underneath the premise.
  6. Turn the strongest assumption into a joke angle, tag, bit, or callback.

Example Input

Premise: Self-checkout machines are getting smarter.

Example Output

Hidden Assumption: Stores assume customers are willing to work for free as long as the machine says thank you.
Comedy Angle: Treat self-checkout like an unpaid job with no training and a judgmental robot supervisor.

Comedy Tip

A hidden assumption is often where the laugh is hiding. Once you expose what everyone accepts without questioning, common sense starts wearing a clown nose.
