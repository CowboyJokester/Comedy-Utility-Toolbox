How To Use This Tool

Purpose

The Callback Search Engine helps you search your saved comedy projects, notes, bits, and setlists to find repeated phrases, themes, tags, callbacks, and forgotten punchlines.

When To Use It

Use this tool when you remember part of a joke but not where you saved it, when you want to find callback opportunities, or when you want to see whether a phrase, character, or theme already appears in your material.

How To Use It

  1. Type a phrase, word, character name, topic, or callback idea into the search box.
  2. Click Search or press Enter.
  3. Review the matching projects, bits, notes, and setlists.
  4. Use the highlighted matches to decide where a phrase or idea can return later.
  5. Click Clear when you want to start a new search.

Example Input

Search Phrase: cargo escort
Other Examples: Grandpa, Aldi, mule, Wally World, two cart lengths, suspicious tractor.

Example Output

The tool lists matching projects, bits, saved setlists, and notes. Each result shows where the phrase was found and highlights the matching text so you can spot callback opportunities quickly.

Comedy Tip

Search for short, repeatable phrases. If a phrase shows up in more than one place, that may be your next callback.
